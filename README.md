# 秘书助手可视化小组件

## 🎯 功能概述
实时显示秘书助手的工作状态，通过办公室场景和像素小人可视化呈现。

## 📱 访问方式

### 方式1：本地访问（推荐）
```bash
# 进入项目目录
cd /root/.openclaw/workspace/widget

# 启动简单HTTP服务器
python3 -m http.server 8080

# 在浏览器访问
http://localhost:8080
```

### 方式2：使用状态API（增强功能）
```bash
# 需要Node.js环境
node status_api.js

# API服务运行在
http://localhost:3000

# 小组件页面
http://localhost:3000
```

### 方式3：直接文件访问
文件位置：`/root/.openclaw/workspace/widget/index.html`

## 🔧 功能特性

### 可视化元素
- 🏢 办公室场景（桌子、电脑）
- 👤 像素小人（多种状态动画）
- 📊 状态面板（实时数据）

### 状态类型
- 💼 工作中（蓝色）
- ⌨️ 打字中（动画）
- 🤔 思考中（左右晃动）
- 🛠️ 开发中（紫色）
- ☕ 休息中（显示咖啡杯）
- 😴 空闲中（灰色）

### 实时数据
- 当前工作状态
- 具体任务内容
- 在线时长
- 最后活跃时间
- 消息处理数量

## 🎮 交互功能
- 点击像素小人切换状态
- 自动状态更新（每10秒）
- 响应式设计（适配手机）

## 📱 三星Z Flip7适配
- 移动端优化布局
- 触摸友好交互
- 适合外屏尺寸显示
